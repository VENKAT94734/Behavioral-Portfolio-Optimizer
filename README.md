### Overview
This project addresses the gap between traditional finance and human psychology. It uses Machine Learning to detect investor biases and modifies **Modern Portfolio Theory (MPT)** to create "Psychologically Optimal" portfolios.

### Key Features
* **Bias Detection:** Random Forest model to identify Overconfidence and Loss Aversion.
* **Algorithmic Adjustment:** Uses `PyPortfolioOpt` to apply L2 Regularization and risk-based constraints based on user profile.
* **Personalized Nudges:** Real-time feedback to guide users toward rational decisions.

### Installation & Usage
1. Clone the repo: `git clone https://github.com/venkat94734/Behavioral-Portfolio-Optimizer.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the platform: `python projectt.py`
